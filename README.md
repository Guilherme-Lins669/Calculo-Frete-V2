# FreteFácil Flask

Projeto convertido de React/TypeScript para Python Flask.

## Como rodar

Instale o Flask:

```bash
pip install -r requirements.txt
```

Execute:

```bash
python app.py
```

Abra no navegador:

```txt
http://127.0.0.1:5000
```

## Estrutura

- app.py -> backend Flask
- templates -> páginas HTML
- static -> CSS
- history.json -> histórico dos cálculos
